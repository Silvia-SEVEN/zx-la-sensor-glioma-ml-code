# zx-la-sensor-glioma-ml-code
# MATLAB Code: Lanthanide-Doped Organic Framework Sensor Array Coupled with Machine Learning for Minimally Invasive Glioma Diagnosis via Cerebrospinal Fluid Biopsy

This repository contains the MATLAB implementation code for all experiments in the paper "Lanthanide-Doped Organic Framework Sensor Array Coupled with Machine Learning for Minimally Invasive Glioma Diagnosis via Cerebrospinal Fluid Biopsy".

## System Requirements
- MATLAB R2020a or later
- Required Toolbox: Statistics and Machine Learning Toolbox

## File Description
Each file is an independent experiment that can be run directly.

## Data Preparation
Place the file in the same directory as the code

## How to Run
Run directly in MATLAB:
```matlab
% Run main experiment
run('exp1_glioma_classification.m')

% Or open the file and click the Run button
